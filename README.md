# resource-cn-web

本仓库用于归档与发布多个独立 HTML 页面。这些页面以静态文件形式存放，可直接在浏览器中打开查看。

## 项目简介

这是一个以 HTML 页面归档为主的仓库，不针对任何特定域名或网站。所有页面均为独立的静态文件，便于保存、分享与离线浏览。仓库内容可能包括信息展示页、工具页面、资源导航页等，具体用途随页面本身而定。

## 目录说明

```
.
├── index.html          # 仓库首页（可选）
├── pages/              # 归档页面存放目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── README.md           # 本文件
```

- `pages/` 目录下存放各个独立 HTML 页面，文件名通常反映页面用途。
- 根目录下的 `index.html` 为可选的仓库入口页面，用于快速浏览或跳转到其他页面。

## 页面归档说明

每个 HTML 页面均为自包含的静态文件，不依赖外部服务器或数据库。页面内容、样式与脚本均嵌入在单一文件中（或引用同一仓库内的资源）。这些页面可能随时间更新或新增，旧版本通常保留在提交历史中。

## 维护说明

- 页面更新：如需修改已有页面，请直接编辑对应 HTML 文件并提交 Pull Request。
- 新增页面：在 `pages/` 目录下添加新文件，并确保文件名简洁明了。
- 删除页面：若某页面不再需要，可将其移除，但建议在提交信息中说明原因。
- 本仓库不提供动态服务或后台接口，所有页面均为纯前端静态内容。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 在浏览器中直接打开 `pages/` 目录下的任意 HTML 文件即可查看。
3. 也可通过 GitHub Pages 或其他静态托管服务部署后在线访问。

## 贡献

欢迎提交 Issue 或 Pull Request 来改进现有页面或增加新内容。请确保所提交的 HTML 文件符合基本规范，且不包含恶意代码或违反平台政策的内容。
